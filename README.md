# ASP.NET Core Web API with NHibernate & JWT Authentication

## 🚀 Overview
This project is a RESTful Web API built using ASP.NET Core. It demonstrates secure authentication, database integration, and clean backend architecture.

## 🔑 Features
- JWT Authentication & Authorization
- Secure Login API (DB-based validation)
- CRUD Operations for Users
- NHibernate ORM with Fluent Mapping
- SQL Server (LocalDB)
- Swagger UI for API testing

## 🛠️ Tech Stack
- ASP.NET Core (.NET 8)
- C#
- NHibernate
- SQL Server (LocalDB)
- JWT (Authentication)

## 📌 API Endpoints
- POST /api/auth/login → Get JWT Token
- GET /api/user → Get all users (Authorized)
- POST /api/user → Create user

## ▶️ How to Run
1. Clone the repo
2. Update connection string (if needed)
3. Run the project
4. Open Swagger: https://localhost:xxxx/swagger
